progress-ring

Lightweight SVG Web Component — zero dependencies, works in plain HTML, Vue 3, and React.

Default

<progress-ring
  value="72"
></progress-ring>

Custom colors & staggered delay

<progress-ring
  value="45"
  primary-color="#e879f9"
  muted-color="#701a75"
  background-color="#140920"
  corner-radius="8"
  animation-delay="200"
></progress-ring>

Animation mode

Both pairs use the same values (25% and 75%). speed mode (default) keeps the visual speed constant. duration mode keeps the clock time constant.

speed (default)
duration 
<progress-ring
  value="25"
  animation-mode="speed"
></progress-ring>

<!-- or lock to a fixed duration -->
<progress-ring
  value="25"
  animation-mode="duration"
></progress-ring>

No animation

<progress-ring
  value="60"
  primary-color="#67e8f9"
  animated="false"
></progress-ring>

Thick ring + auto size

<progress-ring
  value="88"
  primary-color="#86efac"
  muted-color="#1e293b"
  thickness="14"
  size="auto"
></progress-ring>

Interactive

<progress-ring
  id="live"
  value="50"
  primary-color="#e879f9"
  thickness="10"
></progress-ring>
<input type="range" oninput="
  el.setAttribute('value', this.value)
" />

PHP loop

<?php foreach ($tasks as $t): ?>
<progress-ring
  value="<?= $t['percent'] ?>"
  primary-color="#67e8f9"
></progress-ring>
<?php endforeach; ?>

Integer label format

Left uses label-format="integer" → shows 33. Right uses label-format="percent" on the same value → shows 33%.

<progress-ring
  value="33"
  label-format="integer"
></progress-ring>

Format-token label

Any label-format containing { is a template. Available tokens: {value}, {min}, {max}, {percent}.

<progress-ring
  value="3"
  max="10"
  label-format="{value} of {max}"
></progress-ring>

Text override

text-override replaces the label with any string — the arc still reflects value.

<progress-ring
  value="100"
  text-override="✓"
  primary-color="#86efac"
></progress-ring>
